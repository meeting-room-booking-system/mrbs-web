The latest releases will always be available here with a small comment describing what's changed. Just scroll down to the "Latest File Releases" section and click the "download" link in front of "mrbs" package like this :


However I tend to get a bit behind releasing files, so CVS is probably the best way to get stuff. Instructions are here. The stuff actually in CVS at the moment tends to be pretty stable, as I test it a bit before I put it on CVS.

For installation instructions please refer to a file called "INSTALL" in the tarball that you download.
