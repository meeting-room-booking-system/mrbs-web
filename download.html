Getting MRBS
  • Download Tarball
  • CVS Instructions

The latest releases will always be available here with a small comment describing what's changed. Just scroll down to the "Latest File Releases" section and click the "download" link in front of "mrbs" package like this :


Alternatively you can get MRBS via CVS, insructions are here. Currently major development is happening on the trunk, so unless you want to help us testing that, you probably want to check out the 1.2 branch, i.e.: 

      $ cvs -z3 -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/mrbs \
      > co -r mrbs-1_2-branch -P mrbs

For installation instructions please refer to a file called "INSTALL" in the tarball that you download.
